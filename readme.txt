Allows to selectively dequeue script/styles on each post to increase performance of your site
== Description ==

The plugin allows dequeue or remove a script/style on a post basis. It helps to improve performance of the site and you can make sure that your important pages are optimised.



For detailed instructions on using this plugin visit: http://phpwpinfo.com/wordpress-plugin-allows-to-selectively-dequeue-scriptsstyles-on-each-post/




== Installation ==

1. Upload the plugin files in the same way you'd install any other plugin.
2. Activate the plugin
